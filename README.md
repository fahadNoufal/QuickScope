# QuickScope
QuickScope offers a smarter way to stay informed.  Simply enter any topic — from global politics to local trends — and the app scans major social platforms and news outlets in real time. 


## Workflows

1. Update config.yaml
2. Update params.yaml
3. Update entity
4. Update the configuration manager in src config
5. Update the components
6. Update the pipeline
7. Update the main.py
8. Update the app.py
